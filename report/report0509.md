## これから使うであろう用語

* クラウド
    * 雲のような存在、利用者はあまり知ることのない？インターネットのサービス
* software as a service
    * ソフトウェアをサービスとして提供（ブラウザとかで利用できる）
* platform as a service
    * ソフトウェアを実行する環境を提供
* infrastrucuture as a service
    * マシンの貸出（借りて高度な処理をする）
* オンプレミス
    * 自社の設備でソフトウェア運用
* サーバー
    * 物理的なもの、ソフトウェア的なものがある
* API
    * プログラムを書く、実行してくれる奴、コードを受け取る奴
